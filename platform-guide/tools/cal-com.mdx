---

## Setup Cal.com

1. Go to the [Cal.com](https://cal.com/) website and sign up for an account
2. Create a new event type, for example "Meeting with AI assistant"
3. Copy the event ID by clicking on the event type and copying this number from the URL

<img height="200" src="/resources/cal-event-id.png" />

4. Create an API key on Cal.com

## Add the tool to your assistant

Add this tool to your assistant and pass the event ID and API key.
