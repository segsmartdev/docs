Welcome to the Otonistark API documentation. Our API allows you to programmatically interact with the Otonistark platform, enabling you to create and manage leads, schedule calls, and more.

## Base URL

All API requests should be made to the following base URL:

```
https://call.otonistark.com.br/api
```
